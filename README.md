# PulseSurvey

Профессиональная платформа для анкетирования и голосования.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`.

## Функционал

- Конструктор анкет с типами вопросов: текст, одиночный выбор, множественный выбор, рейтинг
- Публикация анкет и сбор ответов
- Агрегированные результаты и визуализация
- Node.js + Express + SQLite

## Деплой на Railway

1. Создайте репозиторий на GitHub и запушьте код.
2. В Railway создайте новый проект `Deploy from GitHub`.
3. Выберите репозиторий и подтвердите автодеплой.
4. Railway использует `npm start` из `package.json`.

Переменные окружения не требуются.
